Welcome to the

Oil Futures ESRA

(Extremely Simple REST API)


This ESRA gives you access to historical monthly oil futures prices from 1980-2021.


INSTRUCTIONS:
To use ESRA, navigate to the /api/daterange endpoint and use a query like the example below to access data:

/api/daterange?name=cushing_1&start=032020&end=072020

This query will return Cushing, OK Crude Oil Future Contract #1 prices for the range March 2020 - July 2020.

Parameters
name e.g. 'cushing_1' (for Cushing, OK Contract #1) or 'harbor_3' (for NY Harbor No. 2 Contract #3)
start start date, formatted MMYYYY
end end date, formatted MMYYYY

To get ALL data from all contracts and all years, use the /api/all endpoint (no parameters).

Note: This API contains data from Cushing, OK Crude Oil Future Contracts 1-3 and NY Harbor No. 2 Heating Oil Future Contracts 1-3.
All prices are in dollars per barrel.
